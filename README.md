# Система лояльности

MVP реферальной системы и Telegram-бота.

## Текущий статус

Подготовлен продуктовый спек и стартовый план реализации.

## MVP Scope

- Telegram-бот для участников и новых клиентов
- Админка в браузере для одного администратора/собственника
- Telegram-уведомления о новых заявках
- Ручное подтверждение бонусов и запросов на списание

## Подготовка к серверу

- проект умеет читать `DATABASE_URL`
- для production поддерживаются `DJANGO_ALLOWED_HOSTS` и `DJANGO_CSRF_TRUSTED_ORIGINS`
- статические файлы Django admin собираются в `backend/staticfiles`
- для запуска на сервере предусмотрен `gunicorn`
